# 12 个月路线图

## M1-M2：基础交易闭环

- 用户、商品、购物车、订单最小闭环
- 统一日志、配置、健康检查
- 本地开发环境标准化（docker-compose）

## M3-M4：交易可靠性

- 库存预占与扣减
- 订单状态机（待支付/已支付/取消/退款中）
- Outbox + 重试队列

## M5-M6：支付与履约

- 支付网关抽象层（多通道）
- 发货、物流轨迹、签收回传
- 售后（退款/退货）流程

## M7-M8：搜索与推荐

- 商品搜索（全文检索 + 聚合筛选）
- 推荐召回与排序（规则 + 模型）
- 首页个性化模块

## M9-M10：运营体系

- 优惠券、满减、秒杀
- 商家后台（商品、库存、订单管理）
- 用户运营（会员、积分）

## M11-M12：平台化与治理

- 服务拆分与弹性扩缩容
- 观测体系（Metrics/Tracing/Alerting）
- 风控与审计（反作弊、黑产识别）
